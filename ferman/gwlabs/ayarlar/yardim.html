Ferman Yardım;

Post Code Nedir ?

Post Code ile veritabanına veri eklemek istediğinizde karşılaşırsınız. Post Code kur.php'yi ilk çalıştırdığınızda karşınıza çıkan formda doldurduğunuz veridir. Post Code'u değiştirmek istediğinizde Güvenlik Kodu'nu bilmeniz gerekecektir. Aksi takdirde Post Code şifrenizi değiştiremezsiniz.

Güvenlik Kodu Nedir ?

Sisteme yönetici eklemek, Post Code değiştirmek için kullanacağınız 3. kademe güvenlik şifresidir. Güvenlik kodunu bilen User Ferman'da her türlü erişime sahiptir. Bu yüzden sisteminizin güvenliği için şifreniz, Post Code'unuz ve Güvenlik Kodunuzun aynı olmamasına özen gösteriniz. Aksi takdirde güvenlik kodunuzdan dolayı sisteminiz hacklenebilir.

Günlüğüm;

Günlük girdilerilerinizi yapacağınız bölümdür. Aylık, yıllık gibi kategorilendirme kalabalık oluşturmaktan başka bir işe yaramadığı için mevcut değildir.

Dökümanlar;

Yazdığınız ya da yayınlamak istediğiniz makale, döküman vb. envai çeşit belgelerinizi yayınlamanız gereken bölüm.

Çalışmalar;

Web aleminde yaptığınız tüm çalışmaları paylaşacağınız bölümünüz.

Bağlantılar;

Arkadaşlarınızın web site adreslerini duyururacağınız ve link vereceğiniz bölüm.

Hakkımda;

Dilediğiniz ölçüde siteyi ziyaret eden misafirlerinizle paylaşacağınız kendinize ait bilgilerin yer aldığı bölüm.

İletişim;

Size ulaşmak isteyenlere msn, mail vb bilgilerinizi sunacağınız bölüm.

Ayarlar;
